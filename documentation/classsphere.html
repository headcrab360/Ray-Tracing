Here we going through sphere and check if rays hit this object.

It hits if there are any t that satisfies equation (P−C)*(P−C) = r^2 where P is any point that ray go though, C is center of sphere.

From sphere in 3D equation: x^2+y^2+z^2=R^2. We want to know if our ray P(t)=A+tb ever hits the sphere anywhere. If it does hit the sphere, there is some t for which P(t) satisfies the sphere equation. So we are looking for any t where this is true: (P(t)−C)*(P(t)−C)=r^2 Full form of P: (A+tb−C)*(A+tb−C)=r^2. And we get t2b*b+2tb*(A−C)+(A−C)*(A−C)−r^2=0.
