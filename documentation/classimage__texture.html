Detailed Description

Texture class that holds an image.

Read in an image and use the 2D (u,v) texture coordinate to index into the image. Texture class that holds an image. Uses texture coordinates instead of image pixel coordinates. These are just some form of fractional position in the image. For example, for pixel (i,j) in an N_x by N_y image, the image texture position is: u = i/(N_x−1) and v = j/(N_y−1).
